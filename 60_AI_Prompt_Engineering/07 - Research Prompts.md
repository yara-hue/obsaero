# Research Prompts

## Phase: Divergence and Ingestion (Week 1)

Use these prompts with the AI specified for each task. Always paste the relevant Rules file system prompt first.

---

### 1. Extract from PDFs
**AI:** NotebookLM
**Task:** Upload an IATA/industry PDF, then run:
```
Extract every statistic, date, and key finding from this document relevant to aviation waste, passenger behavior, and sustainability interventions. Format as a table with: Data point | Source page | Relevance to our pitch | Confidence.
```

### 2. Find Latest Industry Data
**AI:** Perplexity
**Task:**
```
Find the most recent global statistics on airline cabin waste. Include: total waste per passenger, composition breakdown (food, plastic, paper), recycling rates, and year-over-year trends. Prioritize IATA, ICAO, and FAA sources from 2024-2026.
```

### 3. Identify Research Gaps
**AI:** NotebookLM
**Task:** After uploading all gathered sources:
```
Review all uploaded documents and list: 1) Questions we can confidently answer. 2) Important unanswered questions. 3) What specific type of document or data would fill each gap.
```

### 4. Competitive Landscape
**AI:** Perplexity
**Task:**
```
What sustainability initiatives are major airlines currently running for cabin waste reduction? Include: airline name, program name, what they do, measurable results, and timeline. Focus on operational/behavioral interventions (not hardware).
```

### 5. Behavioral Science Foundation
**AI:** Claude
**Task:**
```
Summarize the key principles of choice architecture relevant to waste reduction behaviors. Focus on: default effects, salience, social norms, feedback loops, and friction reduction. For each principle: explain it, give an aviation-relevant example, and rate its applicability to cabin waste (High/Medium/Low).
```

### 6. Synthesize Research Into Briefing
**AI:** Claude
**Task:** Paste your top findings from NotebookLM and Perplexity:
```
Synthesize the following research findings into a 500-word research briefing for the team. Include: the key problem, most surprising statistic, the gap in current solutions, and the most promising behavioral angle for our concept. Write for a team member who has not read the source documents.
```
