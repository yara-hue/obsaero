# Pitch Prompts

## Phase: Pitch Studio (Weeks 3-4)

Use Claude for content drafting, then Gamma for slide building. Always paste the relevant Rules system prompt first.

---

### 1. Narrative Arc
**AI:** Claude
**Task:**
```
Draft an 8-slide narrative arc for our pitch following the Hero's Journey structure:

1. The Ordinary World — what passengers and airlines do today (the waste problem)
2. The Call to Adventure — the surprising data point that reveals the hidden opportunity
3. Refusal of the Call — why current solutions are failing
4. Meeting the Mentor — our behavioral science insight
5. Crossing the Threshold — our system intervention
6. Tests, Allies, Enemies — how it works step by step (including objections and counter-arguments)
7. The Reward — SDG impact and measurable outcomes
8. The Return — call to action and what we need from Boeing

For each slide: title (max 7 words), 3-5 bullet points, suggested visual, emotional goal.
```

### 2. Hook and Opening
**AI:** Claude
**Task:**
```
Write 3 alternative opening hooks for our pitch. Each must:
- Be under 30 seconds spoken
- Start with a memorable image or question
- Reveal a surprising statistic
- Set up the tension between convenience and sustainability

Rate them and recommend the best one with reasoning.
```

### 3. Script Draft
**AI:** Claude
**Task:** Feed the narrative arc from step 1.
```
Write a full spoken script for our 8-slide pitch. Format as:
[SLIDE X: Title]
Spoken lines: [exact words, 30-45 seconds per slide]
Transition cue: [what happens visually between slides]

Rules:
- Conversational, not academic
- Maximum 7 words per headline
- Every slide ends with a hook into the next
- Include pauses marked as (beat)
- Total runtime: 5-6 minutes
```

### 4. Judge Questions Prep
**AI:** Claude
**Task:**
```
Generate 10 likely questions a Boeing judge would ask after our pitch. Categorize as: operational, financial, behavioral evidence, scaling, SDG proof. For each question: write the exact wording a judge would use, then provide our 30-second prepared answer.
```

### 5. One-Pager Executive Summary
**AI:** Claude
**Task:**
```
Write a one-page executive summary of our concept. Structure:
- Problem (2 sentences)
- Our solution (3 sentences)
- How it works (4 bullet points)
- SDG impact (2 sentences)
- Why us (1 sentence)

This will be printed as a leave-behind for judges. Every word counts.
```

### 6. Gamma Slide Content
**AI:** Claude
**Task:** Then paste into Gamma.
```
For each of the 8 slides, provide exactly:
- Title (max 7 words)
- 3-5 bullet points (max 10 words each)
- One data point or statistic
- Suggested visual description

Format as clean text ready to paste into Gamma.
```

### 7. Rehearsal Critique
**AI:** Claude
**Task:**
```
Roleplay as a pitch coach. Review this script for:
1. Clarity — is every idea explained simply enough for a non-expert judge?
2. Pacing — where does it drag? Where should we speed up?
3. Persuasiveness — which arguments are weakest?
4. Time — estimate total runtime. Where can we cut 30 seconds?
5. Landing — does the call to action hit hard enough?
```
