# 🎤 Pitch Studio

## Purpose

Transform the solution into a compelling presentation that sells the idea to Boeing judges.

## Inputs

- The [[Concept Blueprint]]
- Evidence package
- Judge personas from [[Judge Persona Analysis]] (in Winning Strategy)

## Process

1. **Craft the narrative** – use [[Narrative Outline]] to build a hero's journey arc.
2. **Write the script** – in [[Pitch Script]], word‑for‑word with slide cues.
3. **Design visuals** – track assets in [[Visual Assets Tracker]]; generate diagrams with Napkin AI and slides with Gamma.
4. **Rehearse** – use the script for timed runs.

## Outputs

- A polished pitch deck (Gamma link).
- A timed script.
- All visual assets ready for the presentation.

## Files in this Stage

### Narrative Outline
*Purpose:* The emotional and structural story arc for the pitch.

### Pitch Script
*Purpose:* The spoken script, paired with slide transition cues.

### Visual Assets Tracker
*Purpose:* Production pipeline for diagrams, mockups, and graphics.

### Gamma Deck Link
*Purpose:* Live link to the Gamma presentation with version notes.
